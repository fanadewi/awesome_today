# AWESOME TODAY

Awesome Today is a Simple Data collection about Today.

Tools:

* Ruby version '2.5.1'
* Rails version '5.2.2'
* PostgreSql


Configuration:

* bundle install
* bundle exec rails db:setup
* bundle exec rake getter:quote
* bundle exec rails s

Shout out to MuffinLabs for presenting simple data scrapping from Wikipedia!
